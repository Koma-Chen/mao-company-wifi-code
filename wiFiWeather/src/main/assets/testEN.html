Please follow these steps to troubleshoot possible problems and try again

1. Ensure that your equipment has entered the distribution network mode;  

2. Ensure that the input wifi password is correct;  

3. make sure the phone is connected to the wireless router 2.4ghz network;  

4. ensure that the wireless router has turned off the black and white list (MAC address filtering) function;  

5. Please try to set the wireless router to 11bg or 20Mhz bandwidth for distribution network; After the network is successful, reset the router to the original configuration;  

6. if the connected wireless router is a dual-frequency router (2.4ghz /5GHz), make sure the phone is connected to the 2.4ghz network;  

7. no wireless router wireless network name includes Chinese characters;  

8. try to connect other wireless routers to rematch the network;
